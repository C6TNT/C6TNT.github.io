# TNT_JQR GitHub Pages

这是一个适用于 `C6TNT.github.io` 的静态个人站点，包含：

- 个人主页
- 项目作品集
- 博客档案页
- 响应式布局
- 轻量滚动动画

## 文件结构

- `index.html`：首页
- `blog.html`：博客页
- `styles.css`：全站样式
- `script.js`：滚动显现动画
- `.nojekyll`：告诉 GitHub Pages 直接按静态文件发布

## 部署到 GitHub Pages

1. 把这些文件上传到仓库 `C6TNT/C6TNT.github.io`
2. 确保仓库根目录直接包含这些文件
3. 打开 GitHub 仓库设置 `Settings > Pages`
4. 在 `Build and deployment` 中选择：
   - `Source: Deploy from a branch`
   - `Branch: main`
   - `Folder: / (root)`
5. 保存后等待几分钟
6. 访问 `https://c6tnt.github.io/`

## 你下一步最适合替换的内容

- `index.html` 里的个人介绍
- 项目卡片名称、描述和链接
- 联系方式中的邮箱
- `blog.html` 中的示例文章标题和摘要
